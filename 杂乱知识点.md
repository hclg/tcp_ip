# 套接字类型

1. 面向连接 SOCK_STREAM
2. 面向消息 SOCK_DGRAM

# 函数

int socket(int domain, int type,  int protocol); 创建套接字

​	成功返回文件描述符，失败返回-1；
domain 协议族 一般用PF_INFT  ipv4互联网协议族
type 传输类型 

1. 面向连接 SOCK_STREAM

2. 面向信息 SOCK_DGRAM

protocol 协议信息

1. IPPROTO_TCP tcp套接字用于IPv4和面向连接
2. IPPROTO_UDP UDP套接字用于IPv4和面向消息
